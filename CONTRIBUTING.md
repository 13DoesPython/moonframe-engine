MoonFrame is a passion project, and I’d love to see what we can build together! Whether you're fixing a tiny typo in the docs or adding a massive new feature like a particle system, your help is welcome.

How to get involved:

- Got an idea? Open an issue first so we can chat about how to fit it into the engine.

- Found a bug? Let me know! Bonus points if you include a small script that shows the crash.

- Want to code? Fork the repo, make a branch, and send over a Pull Request.

A few "house rules":

- Keep it simple: The goal of MoonFrame is to stay lightweight and easy for beginners.

- Stay compatible: Try to write code that works on most computers without needing the absolute latest hardware.

- Be cool: We’re all here to learn and make games!
